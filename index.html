LostAF

This repository contains a React frontend in /frontend and a FastAPI backend in /backend.

If you see this page on Vercel it means the frontend build either hasn't finished or failed — check your Vercel deployment logs.

Helpful steps:

  • Open the Vercel project > Deployments > Latest to view build logs.
  • Make sure the project either uses the root directory /frontend or the included vercel.json is present.
  • Ensure environment variables (if any) for build are configured in Vercel (e.g. REACT_APP_BACKEND_URL).
